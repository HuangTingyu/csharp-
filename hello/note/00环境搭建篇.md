### 专有名词

```
CLI: Common Language Infrastructure 公共语言基础框架
CLR: Common Language Runtime 公共语言运行时（服务环境）
```



```
CLI：用来处理代码编译的过程，类似JAVA代码编译为字节码的过程，生成.class文件的过程
```



```
CLR：代码运行环境，相当于Java的JVM虚拟机
```
